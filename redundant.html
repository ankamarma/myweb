Cisco Nexus Switch NEX-SW1 Configuration

! --- NEXUS CORE SWITCH NEX-SW1 Configuration ---
version 10.3(9)
hostname nex-sw1
!
feature ospf
feature vpc
!
vpc domain 10
  role priority 1000
  peer-keepalive destination 10.0.0.2 source 10.0.0.1 vrf default
!

Configured Protocols & Network Matrix

  • BGP: Peerings established across boundaries (ASN-62001, ASN-62002, ASN-63001).
  • OSPF: Core backbone routing across edge nodes, ASA firewalls, and Nexus switches.
  • First-Hop Protocols: VRRP tracking at edge paths; HSRP deployed at Nexus core structures.

High Availability (HA) Failover Testing

  • Edge Router Drop: PASS BGP reconverges; VRRP targets failover line seamlessly.
