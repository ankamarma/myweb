Configured Protocols & Network Matrix

  • BGP: Peerings established across ISP boundaries (ASN-62001, ASN-62002, ASN-63001).
  • OSPF: Core backbone routing across edge routers, ASA firewalls, and Nexus core switches.
  • First-Hop Redundancy: VRRP active at edge layers (192.168.10.1); HSRP deployed at Nexus core.
  • vPC & LACP: Implemented across Core layers and FW distribution switches to block loops without Spanning Tree bottlenecks.

High Availability (HA) Failover Testing

  • Edge Router Drop: PASS BGP reconverges; VRRP targets failover line seamlessly.
  • Active Firewall Crash (ASA-FW1): PASS Stateful synchronization over 192.168.100.0/30 updates ASA-FW2 dynamically.
  • Core Nexus Switch Outage: PASS vPC domains keep downstream distribution lines up.
