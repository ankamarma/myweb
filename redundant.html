Edge Router R1 Configuration

! --- Edge Router R1 Configuration ---
hostname R1
!
interface GigabitEthernet1
 description CONNECTS_TO_ISP1
 ip address 192.168.9.1 255.255.255.252
 ip ospf lls disable
 negotiation auto
 no mop enabled
 no mop sysid
!
interface GigabitEthernet2
 description TRANSIT_VLAN10_TO_FW-SW1
 ip address 192.168.10.2 255.255.255.0
 ip ospf priority 50
 ip ospf lls disable
 negotiation auto
 vrrp 10 ip 192.168.10.1
 vrrp 10 priority 150
 vrrp 10 track 1 decrement 60
 no mop enabled
 no mop sysid
!
interface GigabitEthernet3
 ip address 192.168.11.1 255.255.255.252
 ip ospf network point-to-point
 ip ospf lls disable
 ip ospf 1 area 0
 negotiation auto
 no mop enabled
 no mop sysid
!
interface GigabitEthernet4
 no ip address
 shutdown
 negotiation auto
 no mop enabled
 no mop sysid
!
router ospf 1
 router-id 192.168.10.2
 network 192.168.10.0 0.0.0.255 area 0
 default-information originate always
!
router bgp 63001
 bgp log-neighbor-changes
 neighbor 192.168.9.2 remote-as 62001
 neighbor 192.168.9.2 description LINK-TO-ISP1
 !
 address-family ipv4
  network 192.168.10.0
  network 192.168.20.0
  network 192.168.50.0
  network 192.168.60.0
  network 192.168.70.0
  network 192.168.80.0
  neighbor 192.168.9.2 activate
 exit-address-family

Cisco ASA-FW1 Firewall Configuration

: --- ASA-FW1 (ACTIVE) Running Config ---
hostname asa-fw1
mac-address auto prefix 0
!
interface GigabitEthernet0/1
 description LINK-FW-SW
 nameif OUTSIDE
 security-level 0
 ip address 192.168.10.4 255.255.255.0 standby 192.168.10.5
!
interface GigabitEthernet0/3
 description LAN/STATE Failover Interface
!
interface GigabitEthernet0/6
 description LINK-TO-NEX-SW1
 nameif inside
 security-level 100
 ip address 192.168.20.1 255.255.255.0 standby 192.168.20.2

interface Management0/0
 ip address 192.168.90.2 255.255.255.0

 network-object 192.168.80.0 255.255.255.0
 network-object 192.168.20.0 255.255.255.0
!

access-list INSIDE_IN extended permit ip object-group INSIDE_NETWORKS object-group INSIDE_NETWORKS
access-list INSIDE_IN extended permit tcp object-group INSIDE_NETWORKS any eq www
access-list INSIDE_IN extended permit tcp object-group INSIDE_NETWORKS any eq https

Configured Protocols & Network Matrix

  • BGP: Peerings established across boundaries (ASN-62001, ASN-62002, ASN-63001).
  • OSPF: Core backbone routing across edge nodes, ASA firewalls, and Nexus switches.
  • First-Hop Protocols: VRRP tracking at edge paths; HSRP deployed at Nexus core structures.

High Availability (HA) Failover Testing

  • Edge Router Drop: PASS BGP reconverges; VRRP targets failover line seamlessly.
